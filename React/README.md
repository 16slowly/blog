## Command
- 执行 `npm install` 安装依赖
- 执行 `npm run start` 启动 local server，在浏览器中输入 `localhost:3000` 进行本地测试访问
- 执行 `npm run storybook` 启动 storybook 对 UI 组件的测试
- 执行 `npm run build` 进行 prod 环境下的打包
- 执行 `npm run upload` 打包文件将上传至阿里云
